 Review date: 6. May 95                              Review No.:  20
---------------------------------------------------------------------
    Title: Elements                 Game: Doom 2
 Filename: ELEMENTS.ZIP             Size: 40k

   Author: Henry Chang            e-mail: changhenr@urvax.urich.edu

     Maps: MAP01                    Type: normal level
   Sounds: no
 Graphics: no
    Music: no

[ Level construction ] ----------------------------------------------

  Almost every room has near  the same floor level, only a few plat-
forms with  lifts and  a few waste pools.  The whole level  is kinda
'evil', waste, granite, stones. Quite good key-placement, with a few
traps. Nothing new, but a bit different from other PWADs. No special
light usage, mostly average light.
  Look out for secrets near lights...

[ Texture design ] --------------------------------------------------

  Like I mentioned above,  the author used kinda evil textures.  But
only a few  textures  are  misaligned.  Some doors and  skull blocks
(showing the needed key to open the door) are misaligned.

[ Bugs ] ------------------------------------------------------------

  No Tuttie-Frutti, Medusa or HOM effects.

[ Enemies ] ---------------------------------------------------------

  Played in UV only.  Quite hard,  you will get all weapons.  If you
get the blue key, you will know why. Many enemies coming from hidden
rooms, coming out if  you pass  a special line.  Some really  hidden
rooms with Revenants, challenging!

[ Things ] ----------------------------------------------------------

  Enough ammo and health. Enough things at all. Good placed.

[ Scoring ] ---------------------------------------------------------

  72%

  Worth downloading.  Not a fast level, you have to use the map etc.
Solid level with a lot of monsters. Fun!
