[ Level construction ] ----------------------------------------------

  This is another  and probably my  last review of  a VERY bad PWAD.
I don't know why, but I thought  'they did 10 levels, so you have to
review them'... These four guys teach us,  how to create  real ugly,
disgusting and  exorbitantly  illogical  levels.  Their  levels  are
completely weak-willed  drawn and  full of bugs.   Btw. do  you like
levels with almost the same light level? This WAD is a great example
how you should NOT do it.Add to this that many ideas are copied from 
the original levels, boring.

[ Texture design ] --------------------------------------------------

  The  alignment  is  really  great,  because  there  is none.  Bugs 
wherever you go, misalignment on every wall.

[ Bugs ] ------------------------------------------------------------

  A few HOMs, a lot of  Tutti-Frutties.  Medusa?  Some doortraks are
moving, not a single teleporter was well done. Wrong doors etc.

[ Enemies ] ---------------------------------------------------------

  Not very hard, the monsters were randomly placed.

[ Things ] ----------------------------------------------------------

  Enough ammo and health, only a few other things placed. Don't want
to talk about this...

[ Scoring ] ---------------------------------------------------------

  10%

  NOT worth downloading,  I will add  this WAD  to the bad-list too.
10 percent only for being created. These guys really want to do more
levels - HELP!
