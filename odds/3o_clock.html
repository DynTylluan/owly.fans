This is a recording 3o_clock.swf, made by Macop in 2003.

This is a Flash file that I have been aware of since around 2016 when I first downloaded it onto my crapy old laptop, I'm guessing that I must have first found the file on 4chan's /f/ (Flash) board and I must have thought that it was quite cool, so much so that I saved it.

Adobe Flash was discontinued on 2021-01-01, and rather than downloading a doggy, older version of Flash from some shady website, I opted to use ruffle (https://ruffle.rs), a very cool Flash Player emulator built in the Rust programming language that is _very_ easy to use (just drag 'n' drop the .SWF file onto ruffle). The recording was made thanks to me using OBS Studio (https://obsproject.com).

I am not the creator of this movie, but I have uploaded it to YouTube as a way for people to easily view older movies that they might not have known fully about. Please support Macop, their Twitter is https://twitter.com/macop_uwf
